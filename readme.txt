Plugin to load google fonts locally 

== Description ==
Sometimes there are problems loading fonts from google server and it reduces performance of your site. Also as fonts are not loading it may lead to bad user experience.
Local fonts googleapis Divi plugin removes loading of fonts from googleapis site and provides fonts from the local domain. It improves your site performance. It has no settings, just activate it and it starts working immediately.

== Frequently Asked Questions ==

= How it works =

This plugin uses wordpress dequeue and deregister functionality to remove the font styles added by various themes and plugins. And aloow the user to load the fonts locally.

= Does it require any settings =

No! there is no setting for user to do. Its the plug and play Plugin. Just Install and it continue to do its work. In future updates we are planning to add setting section, so that user will have more control on what fonts to replace.
